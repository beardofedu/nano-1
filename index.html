HTML Basics

Basics of the Web

The Web

  • Collection of HTML documents
  • Something like 30 billion pages
  • Invented in the early 1990s

HTML

  • HyperText Markup Language
  • Glues everything together
  • Made up of:
    • Text - what you see
    • Markup - what it looks like
    • Refs to other docs - think videos or images
    • Links to other HTML files - think links to other websites
  • URLs
  • HTTP
  • Web Apps

HTML tags

  • Tags are built with <, contents, and >
  • A tag might be called an element
  • When creating a site and getting display errors, make sure your tags are closed.
    Using a text editor like Atom or Visual Studio Code might help with determining what tags aren't closed

Bold and Italics

  • To create bold or emphasized text use <b> and </b> tags
  • To create italic text use the <i> and </i> tags

Links

  • To find out how to create a link, view the source of this page to see this link.
    • href - identifies where you are linking
    • the words in between <a> and </a> are what will be displayed on the page
